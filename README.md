# 🔍 Selenium Pytest Automation Project

## 📌 Project Overview

This project demonstrates automated testing of a web application using **Python, Selenium, and Pytest**.
It performs search operations on Google and validates the page title.

---

## 🚀 Features

* Automated browser testing using Selenium
* Test execution using Pytest
* Setup & teardown using fixtures (`conftest.py`)
* Organized project structure
* Supports HTML test reports

---

## 🛠️ Technologies Used

* Python
* Selenium
* Pytest
* WebDriver Manager

---

## 📂 Project Structure

```
project/
├── tests/
│   └── test_google.py
├── reports/
├── conftest.py
├── requirements.txt
```

---

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/your-username/your-repo-name.git
```

2. Navigate to project folder:

```
cd your-repo-name
```

3. Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ How to Run Tests

Run all tests:

```
pytest
```

Run tests with HTML report:

```
pytest --html=reports/report.html
```

---

## 🧪 Test Cases Included

* Search "Automation Step by Step" on Google
* Search "Raghav Pal" on Google
* Validate page title

---

## 📊 Test Report

After running tests with `pytest-html`, a report will be generated in:

```
reports/report.html
```
